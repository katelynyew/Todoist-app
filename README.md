##  Overview

A modern task management application inspired by Todoist, enabling users to create, organize, and track tasks across multiple projects. All data persists locally using the browser's Local Storage API.

**Perfect for managing personal projects, daily todos, and long-term goals.**

---

##  Features

✅ **Full CRUD Operations** - Create, read, update, and delete tasks effortlessly  
✅ **Project Organization** - Organize tasks into separate projects  
✅ **Due Dates** - Set and track task deadlines  
✅ **Task Completion** - Mark tasks as complete/incomplete  
✅ **Local Persistence** - Data saved automatically, survives page refreshes  
✅ **Responsive Design** - Works seamlessly on all devices

##  Tech Stack

**Frontend:**
- Vanilla JavaScript (ES6)
- HTML5
- CSS3

**Storage:**
- Local Storage API

**Architecture:**
- Factory Pattern for task/project creation
- Module Pattern for encapsulation
- Event Delegation for performance

##  Getting Started
```bash
# Clone repository
git clone https://github.com/katelynyew/todoist-clone.git

# Navigate to directory
cd todoist-clone

# Open in browser
open index.html
```

No build process required - pure vanilla JavaScript!
